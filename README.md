## Инструкция по использованию приложения

### 1. Перемещение пина на карте
Переместите пин в нужную точку на карте. После этого откроется **bottom sheet** с отображением адреса выбранного места.

### 2. Добавление адреса в избранное
Нажмите кнопку **«Добавить в избранное»**. Откроется модальное окно с подтверждением добавления адреса в избранное.

### 3. Просмотр избранных адресов
Перейдите во вкладку **«Избранные»** в нижнем таб-баре. Откроется экран со списком ранее сохранённых адресов.

### 4. Удаление адреса из избранного
Для удаления адреса выполните **долгое нажатие** на нужный элемент в списке. Адрес будет удалён из избранного.

### 5. Поиск адреса на карте
На экране с Яндекс.Картой нажмите кнопку **«Поиск»**. Откроется **bottom sheet** с полем для ввода адреса.

- Введите нужный адрес.
- Появится список подходящих результатов.
- Выберите нужный адрес из списка.

После выбора адрес отобразится на карте, и откроется **bottom sheet** с названием выбранного места.
